# Community Guidelines

This project aims to keep collaboration respectful, constructive, and focused on
technical quality.

## Expected Behavior

- Be respectful and professional in issues, pull requests, and discussions.
- Give constructive feedback on code and design, not on people.
- Assume good intent and ask clarifying questions before escalating.
- Keep discussions on-topic and actionable.

## Unacceptable Behavior

- Harassment, discrimination, hate speech, or threats.
- Insults, personal attacks, or repeated hostile behavior.
- Sharing private information without permission.
- Spam, trolling, or intentionally disruptive conduct.

## Scope

These guidelines apply to all project spaces, including issues, pull requests,
discussions, reviews, and any other repository communication channels.

## Enforcement

Maintainers may take action including:

1. Warning
2. Temporary restriction
3. Removal of content or permanent ban

Actions are case-by-case, with the goal of keeping the project safe and
productive.

## Reference

This document is a lightweight policy inspired by
[Contributor Covenant](https://www.contributor-covenant.org/).
